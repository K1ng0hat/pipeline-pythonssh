Python SSH Pipeline <br>
</b>Overview</b>

This project is a Python-based automation pipeline designed to connect to a Linux server via SSH and execute structured, repeatable tasks such as health checks, deployments, maintenance operations, and service management.

The pipeline is modular, maintainable, and aligned with real-world DevOps and CI/CD practices, using Python as the orchestration layer and SSH as the remote execution mechanism.

<b>Key Features</b>

- Secure SSH connection using key-based authentication

- Modular pipeline architecture

- Centralized SSH client management

- Controlled execution of remote commands

- Proper error handling using exit codes

- Ready to be extended into CI/CD environments
